# Prática 5 – Parte 1 e 2

Este repositório contém os códigos blink.py, blink1.py e o serviço blink.service implementados na Raspberry Pi. O projeto consiste em controlar um LED via GPIO e configurar um serviço systemd que executa um script automaticamente ao iniciar a raspi. Quando se para o serviço, ela executa outro comando, que é o blink mais rápido, e para o primeiro, blink mais lento.
